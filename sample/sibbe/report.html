3D Tiles Styling Plugin — Sibbe Sample

This sample demonstrates the CesiumStylingPlugin for 3DTilesRendererJS, which enables Cesium 3D Tiles Styling specification for coloring and styling 3D Tiles based on feature properties.

Live Viewer

The viewer below displays BAG (Basisregistratie Adressen en Gebouwen) buildings in Sibbe, Limburg, Netherlands, colored by building height using a gradient from dark navy to vivid rose.

About This Sample

Styling Configuration

The plugin uses the height property from each building feature to apply a color gradient. The style configuration uses Cesium's conditional styling syntax: 

{
  style: {
    color: {
      conditions: [
        ["${feature['height']} <= 5", "color('#1a1a2e')"],   // Deep navy (0-5m)
        ["${feature['height']} <= 10", "color('#16213e')"],  // Dark blue (5-10m)
        ["${feature['height']} <= 15", "color('#0f3460')"],  // Medium blue (10-15m)
        ["${feature['height']} <= 20", "color('#533483')"],  // Purple (15-20m)
        ["${feature['height']} > 20", "color('#e94560')"],   // Vivid rose (>20m)
        ["true", "color('#ffffff')"]                      // Fallback white
      ]
    }
  }
}

Dataset Information

Source: pg2b3dm tool (PostgreSQL to 3D Tiles converter)

Data: BAG buildings in Limburg, Netherlands (Sibbe area)

Tiling: Implicit QUADTREE with 3 GLB content files

Properties:

  • identificatie — Building ID (string)
  • height — Building height in meters (double)

Technical Stack

  • MapLibre GL JS: Base map rendering and camera control
  • Three.js: 3D graphics rendering engine
  • 3DTilesRendererJS: 3D Tiles loader and renderer
  • CesiumStylingPlugin: Feature-based styling using Cesium 3D Tiles Styling spec

How It Works

The plugin registers as a 3DTilesRendererJS plugin and applies material overrides to each mesh based on feature metadata. The styling expressions are evaluated per-feature using the EXT_mesh_features and EXT_structural_metadata glTF extensions.

Required GLTF loader extensions:

  • GLTFMeshFeaturesExtension — Parses mesh-level feature IDs
  • GLTFStructuralMetadataExtension — Parses feature property tables
  • GLTFCesiumRTCExtension — Handles relative-to-center positioning
