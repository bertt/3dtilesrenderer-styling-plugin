## Project Context

- **Owner:** Bert Temme
- **Project:** 3dtilesrenderer-styling-plugin — A Cesium 3D Tiles Styling Specification plugin for 3DTilesRendererJS. Implements color conditions based on feature attributes (e.g. height), mapped to Three.js materials and GLSL shaders. Includes a MapLibre GL sample viewer for BAG buildings in Limburg (Sibbe area), npm package publication at @bertt/3dtilesrenderer-styling-plugin, README documentation, and a blog post.
- **Stack:** JavaScript (ES modules), Three.js, GLSL, MapLibre GL JS, 3DTilesRendererJS, npm, pg2b3dm (3D Tiles source), Maptiler terrain
- **Universe:** The Matrix
- **Created:** 2026-03-27
