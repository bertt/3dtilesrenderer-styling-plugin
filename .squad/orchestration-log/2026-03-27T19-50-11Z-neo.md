### Key Decisions Implemented

1. **Vertex Color Approach** — Applied vertex colors to mesh geometry based on per-feature conditions
   - Leverages `_FEATURE_ID_0` attribute from EXT_mesh_features
   - No custom shaders required (works with standard Three.js materials)
   - Single geometry draw call per mesh (efficient)

2. **Expression Evaluator** — Regex + Function constructor
   - Replaces `${feature['property']}` tokens with actual values
   - Normalizes operators (`==` → `===`)
   - Safe execution in strict mode
   - No external dependencies (CDN-friendly)

3. **Package Structure** — Zero-dependency ES module
   - Peer dependencies prevent version conflicts
   - Works with any compatible three.js and 3d-tiles-renderer
   - CDN-friendly via esm.sh
